# Energy Emission Dashboard SaaS

## Struktur Proyek
- index.html : Dashboard premium (frontend)
- style.css  : Styling dark premium
- script.js  : JS untuk Supabase + charts + filters
- README.md  : Dokumentasi singkat

## Deploy
1. Buat akun Vercel: https://vercel.com
2. Siapkan folder proyek ini
3. Deploy folder ini di Vercel → URL publik langsung
4. Pastikan Supabase ANON KEY sudah bisa SELECT data dengan RLS aktif
